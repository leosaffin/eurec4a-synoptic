This repository contains the python code to reproduce the plots for the paper "A synoptic overview of the EUREC<sup>4</sup>A ﬁeld
campaign" currently in review for QJRMetS

It also contains extra plots.
- The folder `synoptic_charts` contains the synoptic charts produced by Météo-France Antilles-Guyane during the field campaign. Note that some days are missing because they were not produced on some days.
- The folder `snapshots` contains a copy of the snapshot figures from the paper for each day
